# Installation

Get the code

`git clone git@github.com:greg-landing/yamnet.git`

`cd yamnet`


Create a virtual environment

`python3 -m venv environment`
`source environment/bin/activate`

Install requirements

`pip install -r requirements.txt`

Run it

`python inference.py`
